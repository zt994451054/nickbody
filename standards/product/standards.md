# 产品领域规范

> 本规范优先于公司级规范，未覆盖条目以公司级规范为准。

## 业务逻辑约束

1. **软惩罚原则**：健康分可衰减（有下限、不清零），**段位永不下降**；任何新功能设计不得引入"欠债感"机制
2. **绝不强制原则**：不锁屏、不模态阻断、不打断心流；打断性交互只能由宠物情绪表达承载
3. **慷慨免费原则**：免费用户必须能体验完整核心循环并将宠物养到 Gold；付费只卖"更高情绪价值密度"
4. **隐私第一原则**：摄像头画面永不显示、永不存储、永不上传；跟练界面常驻隐私声明

## 功能边界

永久排除（红线）：工作监控/生产力统计类功能、医疗诊断与治疗宣称、现有 IP 形象的未授权使用。
V1 排除（规划中）：多宠物货架、智能疲劳值、社交/排行榜、账号云同步、Windows/移动端。
