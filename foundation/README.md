# 项目基础文档

> 本目录存放跨版本的长期文档，是产品的「底座知识」。不随版本迭代消失，代表当前最新全貌。

## 文档导航

| 目录 | 说明 |
|------|------|
| [market/](./market/) | 市场调研、竞品分析、用户研究 |
| [business/](./business/) | 商业模式、上市策略 |
| [product-arch/](./product-arch/) | 产品架构（功能模块/用户旅程，持续更新） |
| [tech-arch/](./tech-arch/) | 技术架构（系统架构/技术决策，持续更新） |

## 更新原则

- **market/** 和 **business/**：立项时完成，后续按需补充
- **product-arch/** 和 **tech-arch/**：每次版本迭代后如有架构变更，在对应 changelog.md 追加记录
