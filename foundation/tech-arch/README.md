# 技术架构说明

> 本文档代表技术架构的当前最新全貌，随技术演进持续更新。

## 文档导航

| 文档 | 说明 |
|------|------|
| [overview.md](./overview.md) | 系统架构全貌（技术栈/架构图/模块/安全/性能）|
| [decisions/](./decisions/) | 技术决策记录（ADR）|
| [changelog.md](./changelog.md) | 架构变更历史 |

## 更新原则

- overview.md 代表当前最新全貌，不记录历史
- 每次版本迭代涉及架构调整时，在 changelog.md 追加记录
- 重大技术决策须在 decisions/ 下创建对应 ADR 文件
